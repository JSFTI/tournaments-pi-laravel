Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost:8000

Authenticating requests

This API is not authenticated.

Manage Brackets

API end-poitns to manage tournament brackets.

Create new brackets

Brackets will be filled randomly with players associated with the tournament. Brackets will be build until the final bracket.

Example request:
curl --request POST \
    "http://localhost:8000/tournaments/5/brackets?enum(tree,list)=doloribus" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8000/tournaments/5/brackets"
);

const params = {
    "enum(tree,list)": "doloribus",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "POST",
    headers,
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://localhost:8000/tournaments/5/brackets',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'query' => [
            'enum(tree,list)'=> 'doloribus',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Request      

POST tournaments/{tournament}/brackets

URL Parameters

tournament  integer  

Query Parameters

enum(tree,list)  string optional  

Defaults to "tree". Returns created brackets in tree or list.
For trees, brackets will be structured recursively in a binary tree, under the prev_match attribute with the last match acting as root.
For lists, brackets will be structured in an array.

Response

Response Fields

brackets  object[]  

Array of bracket objects.

brackets.match  integer  

Consider this the identifier of a bracket in a tournament.

brackets.player  object  

Player in this bracket. If null, it means that the bracket has no player yet.

brackets._url  string  

URL to bracket.

brackets.prev_match  object  

Previous match. Seperated to left and right side. If null, it means that the bracket has no previous match.

brackets.prev_match.left  object  

Prevous match on the left side.

brackets.prev_match.right  object  

Prevous match on the right side.

_url  string  

URL to bracket list of the tournament.

tournament_url  string  

URL to the tournament.
