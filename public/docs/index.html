Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost:8000

Authenticating requests

This API is not authenticated.

Endpoints

Request      

GET tournaments/{tournament}/brackets

URL Parameters

tournament  integer  

Query Parameters

dataStructure  string optional  

enum(tree,list) Defaults to "list". Returns created brackets in tree or list.
For trees, brackets will be structured recursively in a binary tree, under the prev_match attribute with the last match as root.
For lists, brackets will be structured in an array.

Create new brackets or replace old brackets.

requires authentication

Brackets will be filled randomly with players associated with the tournament. Brackets will be build until the final bracket. This endpoint can be called again to randomly insert recently added players into the brackets.
Only tournament owner can create or replace brackets.

Calling this endpoint replaces previously generated bracket.
No body parameters are required for this end-point. Any body parameters will be ignored
Example request:

Upsert Player in Bracket

requires authentication

Insert a player to a bracket or replace a player in a bracket. Upsert can only be done if tournament has not started yet.

If upserted player is already in the tournament brackets and the bracket is already assigned to a player, then both players' position in the bracket will be swapped.
Example request:
curl --request PUT \
    "http://localhost:8000/brackets/5/player" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"player_id\": 20
}"

const url = new URL(
    "http://localhost:8000/brackets/5/player"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "player_id": 20
};

fetch(url, {
    method: "PUT",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->put(
    'http://localhost:8000/brackets/5/player',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'player_id' => 20,
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
